# daytona2026

## Brief Description
Build a consumer “paste-the-message” scam checker that renders a plain-English
verdict AND runs a live “detonation engine”: a Daytona sandbox drives a headless
Chromium browser through an Oxylabs Singapore residential proxy to defeat the
phishing kit’s geo-cloaking, screenshots the fake Telegram login page, and an ai&
hosted LLM (plus a Nosana-hosted open vision model) explains exactly how the
page would hijack your Telegram account and spread to your contacts.
